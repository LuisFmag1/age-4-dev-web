Explicação do projeto

Este projeto foi desenvolvido com o intuito de treinar a manipulação de arrays e objetos. O projeto consiste em uma tabela que exibe informações sobre civilizações de um jogo de estratégia. A tabela é preenchida com informações de um array de objetos, que contém informações sobre cada civilização.
